**treesitter.run** is an alternative web playground for tree-sitter, the
parser generator tool and incremental parsing library.

It supports multiple languages, and uses the official
[web-tree-sitter](https://www.npmjs.com/package/web-tree-sitter){target="\_blank"}
library to load in
[WASM files](https://github.com/terror/treesitter.run/tree/master/public){target="\_blank"}
for the various languages it supports.
